Bootstrap 進階網格系統應用

1. 巢狀網格系統

在欄位中可以再次使用網格系統，創建更複雜的布局。巢狀網格也遵循12欄位系統的規則。
巢狀欄位 1
巢狀欄位 2
主要欄位

2. 欄位排序

使用 order 類別來改變欄位的視覺順序，而不改變 HTML 結構。這對於響應式設計特別有用。
1. HTML中第一個 (顯示第三)
2. HTML中第二個 (顯示第一)
3. HTML中第三個 (顯示第二)

3. 自動佈局欄位

使用 col-auto 讓欄位根據內容自動調整寬度，非常適合處理不確定內容長度的情況。
短文字
較長的文字內容
非常長的文字內容展示

4. Gutters 間距控制

使用 g-* 類別控制欄位間的間距。數字越大，間距越大（範圍：0-5）。
大間距欄位 1
大間距欄位 2
小間距欄位 1
小間距欄位 2

5. 混合響應式設計

結合多種響應式類別實現複雜的布局變化。隨著螢幕尺寸變化，欄位會自動調整其寬度。

xs: 100%

sm: 50%

md: 33.33%

lg: 25%

xl: 16.66%
