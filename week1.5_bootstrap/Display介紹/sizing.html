Bootstrap Sizing 尺寸控制詳解

1. 寬度控制 (Width)

寬度類別說明：
w-{size}: 25%, 50%, 75%, 100%, auto
size 數值對應百分比：25=25%, 50=50%, 75=75%, 100=100%
w-25 (寬度 25%)
w-50 (寬度 50%)
w-75 (寬度 75%)
w-100 (寬度 100%)
w-auto (自動寬度)

2. 高度控制 (Height)

高度類別說明：
h-{size}: 25%, 50%, 75%, 100%, auto
vh-100: 視窗高度 100%
h-25
h-50
h-75
h-100

3. 相對於父元素的尺寸控制

相對尺寸類別說明：
mw-100: 最大寬度 100%
mh-100: 最大高度 100%
min-vw-100: 最小視窗寬度 100%
最大高度 100%

4. 視窗相對尺寸 (Viewport)

視窗尺寸類別說明：
vw-100: 視窗寬度 100%
vh-100: 視窗高度 100%
min-vw-100: 最小視窗寬度
min-vh-100: 最小視窗高度
vw-100 (視窗寬度 100%)
一般容器寬度 (對比用)

5. 響應式尺寸控制

響應式尺寸技巧：
- 結合媒體查詢斷點
- 使用 flex 和 grid 系統
- 配合容器類別使用
響應式寬度： 小螢幕 100%， 中螢幕 75%， 大螢幕 50%

6. 實用組合範例

固定寬度圖片卡片

使用 mw-100 確保圖片不會超出容器

自適應高度卡片

使用 h-100 讓卡片高度自適應

這是額外的內容來展示高度差異

全屏背景效果

7. 尺寸控制最佳實踐

重要提醒：
  • 優先使用相對單位（%, vh, vw）而非固定單位
  • 考慮響應式設計需求
  • 注意內容溢出問題
  • 合理使用最大/最小尺寸限制
  • 配合 flex 和 grid 系統使用
