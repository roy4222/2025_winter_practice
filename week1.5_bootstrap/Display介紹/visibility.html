Bootstrap Visibility 可見性控制詳解

1. 基本顯示控制

基本顯示類別：
- visible：設置元素為可見
- invisible：隱藏元素但保留空間
- d-none：完全隱藏元素且不保留空間
這是一個可見的元素 (visible)
這個元素被隱藏但保留空間 (invisible)
這個元素完全隱藏且不保留空間 (d-none)

2. 響應式顯示控制

響應式顯示類別命名規則：
d-{breakpoint}-{value}
斷點 (breakpoint): sm, md, lg, xl, xxl
值 (value): none, block, inline, inline-block 等
只在中等尺寸(md)以上的螢幕顯示
只在大螢幕(lg)以下顯示
只在小螢幕(sm)顯示

3. 列印可見性控制

列印相關類別：
- d-print-none：列印時隱藏
- d-print-block：列印時顯示為區塊
- d-none d-print-block：螢幕上隱藏但列印時顯示
這個內容只在螢幕上顯示，列印時會隱藏
這個內容在螢幕上隱藏，但列印時會顯示

4. 顯示方式控制

顯示方式類別：
- d-block：區塊顯示
- d-inline：行內顯示
- d-inline-block：行內區塊顯示
- d-flex：彈性盒子顯示
這是區塊顯示 (d-block) 這是行內顯示 (d-inline) 這也是行內顯示
這是行內區塊顯示 (d-inline-block)

6. 可見性最佳實踐

重要提醒：
  • 使用 d-none 會完全移除元素，適合需要節省空間的場景
  • 使用 invisible 會保留元素空間，適合需要保持頁面布局的場景
  • 響應式類別可以組合使用，但要注意優先級
  • 考慮使用 aria-* 屬性來提升無障礙性
