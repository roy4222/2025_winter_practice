Bootstrap Spacing 間隔工具詳解

1. Margin（外邊距）完整示例

Margin 類別命名規則：
m{方向}-{大小}
方向: t(top), b(bottom), s(start), e(end), x(左右), y(上下)
大小: 0-5, auto
m-3：四個方向都有 margin (1rem = 16px)
mb-3：底部 margin
ms-4：左側 margin (1.5rem = 24px)
mx-auto：水平置中

2. Padding（內邊距）完整示例

Padding 類別命名規則：
p{方向}-{大小}
方向: t(top), b(bottom), s(start), e(end), x(左右), y(上下)
大小: 0-5
p-4：四周 padding (1.5rem)
px-5：左右 padding (3rem)
py-3：上下 padding (1rem)

3. 響應式間隔設計

響應式類別命名規則：
{屬性}{方向}-{斷點}-{大小}
斷點: sm, md, lg, xl, xxl
響應式 margin：
小螢幕：m-2 (0.5rem)
中螢幕：m-md-4 (1.5rem)
大螢幕：m-lg-5 (3rem)
響應式 padding：
隨螢幕寬度變化而改變內距

6. 間隔數值參考表

尺寸類別 rem 值 像素值 常用場景
0 0 0px 移除間距
1 0.25rem 4px 極小間距
2 0.5rem 8px 小間距
3 1rem 16px 中等間距
4 1.5rem 24px 大間距
5 3rem 48px 特大間距
auto auto auto 自動間距（常用於水平置中）
